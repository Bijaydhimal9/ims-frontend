

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation


1. Install dependencies:
```bash
npm install
```

2. Set up environment variables:
```bash
cp .env.example .env
```
Edit `.env` with your API keys and configuration.

### Running the App

```bash
npm run dev
```

## Tech Stack
- React
- TypeScript
- Tailwind CSS
